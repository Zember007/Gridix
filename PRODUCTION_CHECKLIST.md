## 🚨 Исправление ошибки "Unauthorized" при загрузке фотографий:

Если после добавления регистрации не удается загружать фотографии:

1. **Выполните SQL скрипт** `docs/STORAGE_FIX.sql` в Supabase Dashboard
2. **Убедитесь что пользователь авторизован** перед загрузкой
3. **Проверьте что bucket `project-images` существует** и настроен как public
4. **Проверьте политики Storage** - должны быть созданы 4 политики для project-images
5. **Проверьте RLS политики для таблиц** - должны быть созданы политики для layout_photos и apartment_photos

### Распространенные ошибки:
- `Unauthorized` при загрузке файлов → проблема с Storage политиками
- `new row violates row-level security policy` → проблема с RLS политиками таблиц
- `403 Forbidden` → пользователь не авторизован или политики настроены неправильно

## 🔗 Новые embed URL:

- `/embed/projects/USER_ID` - только проекты пользователя USER_ID
- `/embed/projects-map/USER_ID` - только проекты пользователя USER_ID на карте

### С выбором языка по умолчанию:
- `/embed/projects/USER_ID?lang=ru` - проекты пользователя на русском языке

- `/embed/project/PROJECT_ID?lang=ru` - конкретный проект на русском языке

Где `USER_ID` - это UUID пользователя из таблицы auth.users
