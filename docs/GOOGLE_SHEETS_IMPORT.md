# Импорт данных из Google Sheets

## Обзор

Система поддерживает импорт данных проектов напрямую из Google Sheets документов. Это решает проблемы с некорректными ссылками и обеспечивает стабильную синхронизацию данных.

## Основные возможности

### ✅ Что улучшено

1. **Автоматическое преобразование URL** - любой формат ссылки Google Sheets автоматически преобразуется в правильный формат для экспорта
2. **Интеллектуальное распознавание** - система автоматически определяет тип ссылки и адаптирует интерфейс
3. **Валидация доступа** - проверка доступности документа перед импортом
4. **Динамический UI** - интерфейс адаптируется под тип импортируемого документа
5. **Подробные инструкции** - пошаговые инструкции для настройки доступа к Google Sheets

### 📋 Поддерживаемые форматы URL

Система корректно обрабатывает все форматы Google Sheets URL:

```
// Стандартный URL с gid
https://docs.google.com/spreadsheets/d/{ID}/edit#gid={SHEET_ID}

// URL без gid (используется первый лист)
https://docs.google.com/spreadsheets/d/{ID}/edit

// URL просмотра
https://docs.google.com/spreadsheets/d/{ID}/view

// Короткий формат
https://docs.google.com/spreadsheets/d/{ID}
```

Все эти форматы автоматически преобразуются в:
```
https://docs.google.com/spreadsheets/d/{ID}/export?format=xlsx&gid={SHEET_ID}
```

## Как использовать

### 1. Настройка Google Sheets документа

1. Откройте ваш Google Sheets документ
2. Нажмите кнопку **"Поделиться"** в правом верхнем углу
3. В разделе **"Общий доступ"** выберите **"Доступен всем в интернете"**
4. Убедитесь, что установлен режим **"Читатель"**
5. Скопируйте ссылку на документ

### 2. Импорт через интерфейс

1. Перейдите в **"Создание проекта"** → **"Импорт из Excel"** → **"По ссылке"**
2. Вставьте ссылку на Google Sheets документ
3. Система автоматически:
   - Определит, что это Google Sheets
   - Изменит интерфейс под Google Sheets
   - Покажет соответствующие инструкции
4. Нажмите **"Проверить"** для валидации доступа
5. При успешной проверке нажмите **"Импортировать данные"**

### 3. Требования к структуре данных

- **Первая строка** должна содержать заголовки столбцов
- **Обязательные поля**: номера квартир, этажи, комнаты, площадь, цена, статус
- Данные должны начинаться со **второй строки**

## Технические детали

### Компоненты

1. **`/src/lib/google-sheets-utils.ts`** - утилиты для работы с Google Sheets
2. **`/src/components/ExcelUrlImporter.tsx`** - обновленный компонент импорта
3. **`/src/components/GoogleSheetsImportDemo.tsx`** - демонстрационный компонент
4. **`/src/pages/GoogleSheetsTestPage.tsx`** - страница тестирования

### Функции в google-sheets-utils.ts

```typescript
// Преобразование URL в формат экспорта
convertGoogleSheetsUrl(url: string): string

// Проверка, является ли URL ссылкой на Google Sheets
isGoogleSheetsUrl(url: string): boolean

// Парсинг информации из URL
parseGoogleSheetsUrl(url: string): { documentId, sheetId, isValid }

// Валидация доступности документа
validateGoogleSheetsAccess(url: string): Promise<ValidationResult>

// Получение инструкций для пользователей
getGoogleSheetsInstructions(): InstructionsData
```

### Обработка ошибок

Система обрабатывает следующие ошибки:

- **403 Forbidden** - документ недоступен для публичного доступа
- **404 Not Found** - документ не существует или ссылка некорректна
- **Network errors** - проблемы с сетью
- **Invalid URL** - некорректный формат ссылки Google Sheets

## Преимущества нового подхода

1. **Надежность** - автоматическое преобразование исключает ошибки пользователя
2. **Простота** - пользователь может вставить любую ссылку Google Sheets
3. **Интуитивность** - интерфейс адаптируется под тип документа
4. **Синхронизация** - данные остаются актуальными при изменениях в документе
5. **Валидация** - проверка доступности перед импортом предотвращает ошибки
