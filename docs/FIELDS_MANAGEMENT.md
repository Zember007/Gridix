# Управление полями проекта

## Обзор

Система управления полями позволяет настраивать порядок отображения и видимость всех полей квартир в проекте, включая как стандартные, так и кастомные поля.

## Возможности

### 1. Управление стандартными полями
- **Номер квартиры** - текстовое поле
- **Этаж** - числовое поле  
- **Комнаты** - числовое поле
- **Площадь (м²)** - числовое поле
- **Цена** - числовое поле
- **Статус** - выпадающий список (доступно/забронировано/продано)

### 2. Управление кастомными полями
- **Текстовые поля** - для ввода текста
- **Числовые поля** - для ввода чисел
- **Выпадающие списки** - для выбора из предустановленных вариантов
- **Переключатели** - для булевых значений (да/нет)

### 3. Настройка отображения
- **Перестановка полей** - drag & drop для изменения порядка
- **Управление видимостью** - скрытие/показ полей через переключатель
- **Визуальное разделение** - стандартные и кастомные поля помечены бейджами

## Как использовать

### Доступ к управлению полями
1. Откройте проект в редакторе
2. Перейдите на вкладку **"Поля"**
3. Здесь вы увидите все поля проекта

### Изменение порядка полей
1. Наведите курсор на поле
2. Захватите поле за иконку ≡ (drag handle)
3. Перетащите поле в нужное место
4. Порядок автоматически сохранится

### Управление видимостью
1. Нажмите на иконку глаза рядом с полем
2. 👁️ - поле видимо в проекте
3. 👁️‍🗨️ - поле скрыто (затемнено)

### Добавление кастомных полей
1. Нажмите кнопку **"Добавить кастомное поле"**
2. Заполните форму:
   - Название поля
   - Тип поля
   - Параметры (для списков)
   - Обязательность
3. Сохраните поле

### Редактирование кастомных полей
1. Нажмите иконку редактирования ✏️ рядом с кастомным полем
2. Внесите изменения
3. Сохраните

## Ограничения

- **Стандартные поля** нельзя удалять или редактировать (только скрывать/показывать и менять порядок)
- **Кастомные поля** можно полностью редактировать и удалять
- Порядок полей применяется ко всему проекту

## Защита от конфликтов

Система автоматически защищает от одновременных изменений:

- **Во время сохранения** все операции блокируются (перетаскивание, редактирование, удаление)
- **Визуальная индикация** - поля становятся полупрозрачными с курсором "недоступно"
- **Спиннер загрузки** в заголовке показывает активность сохранения
- **Автоматическая разблокировка** после завершения операции

## Техническая реализация

### База данных
- `project_field_settings` - настройки стандартных полей
- `project_custom_fields` - кастомные поля (расширены полями `sort_order`, `is_visible`)

### Компоненты
- `AllFieldsManager` - основной компонент управления полями
- `ApartmentDetailsWithFields` - отображение полей в квартирах
- `CustomFieldsManager` - редактирование кастомных полей

### Миграции
- `20250125000000-add-field-ordering.sql` - добавление полей сортировки
- `20250125100000-add-project-field-settings.sql` - система управления всеми полями
