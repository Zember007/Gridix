# Gridix FloorPlan Wizard - Полное описание функционала

## 📋 Общее описание сервиса

**Gridix FloorPlan Wizard** — это инновационная веб-платформа для создания интерактивных планов недвижимости. Сервис позволяет застройщикам и риелторам создавать интерактивные планы этажей, управлять информацией о квартирах, интегрироваться с CRM-системами и встраивать виджеты на сайты.

### 🎯 Основные цели сервиса:
- Упрощение процесса продаж недвижимости
- Создание интерактивных планов этажей
- Автоматизация управления проектами
- Интеграция с CRM-системами (AmoCRM)
- Предоставление готовых виджетов для сайтов

---

## 🏗️ Архитектура и технологии

### Frontend Stack:
- **React 18** с TypeScript
- **Vite** как сборщик
- **Tailwind CSS** для стилизации
- **shadcn/ui** компоненты
- **React Router** для навигации
- **TanStack Query** для управления состоянием
- **Leaflet** для карт
- **React Hook Form** для форм
- **Recharts** для графиков

### Backend Stack:
- **Supabase** как Backend-as-a-Service
- **PostgreSQL** база данных
- **Row Level Security (RLS)** для безопасности
- **Edge Functions** для серверной логики
- **Real-time subscriptions** для живых обновлений

### Интеграции:
- **AmoCRM API** для управления лидами
- **Google Sheets API** для импорта данных
- **Excel файлы** (xlsx, xls, csv)
- **PDF генерация** для отчетов
- **Leaflet Maps** для геолокации

## 🎨 Система визуализации

### 🏢 Режимы просмотра:

#### **1. Фасад здания:**
- Интерактивный план всего здания
- Выделение этажей при наведении
- Переход к детальному плану этажа
- Статистика по этажам

#### **2. План этажа:**
- Детальный план конкретного этажа
- Интерактивные полигоны квартир
- Отображение номеров квартир
- Цветовое кодирование статусов

#### **3. Список квартир:**
- Табличное представление
- Фильтрация по параметрам
- Сортировка по полям
- Поиск по номеру/характеристикам

#### **4. Карта проектов:**
- Геолокация проектов на карте
- Фильтрация по регионам
- Информационные попапы
- Навигация к проектам

### 🎯 Интерактивность:
- **Клик по квартире** - детальная информация
- **Наведение** - краткая информация
- **Фильтры** по статусу, комнатности, цене
- **Поиск** по номеру квартиры
