# Конфигурация цветовой схемы админ панели

Этот файл описывает, как изменить цветовую схему админ панели в одном месте.

## Основной файл конфигурации

Все цвета админ панели настраиваются в файле: `src/lib/admin-theme-config.ts`

## Как изменить цветовую схему

### 1. Быстрая смена темы

Чтобы переключиться между предустановленными темами, измените экспорт `ADMIN_THEME`:

```typescript
// Для черной темы (текущая)
export const ADMIN_THEME: AdminThemeConfig = blackAdminTheme;

// Для синей темы (оригинальная)
export const ADMIN_THEME: AdminThemeConfig = blueAdminTheme;
```

### 2. Создание собственной темы

Создайте новый объект темы:

```typescript
export const customAdminTheme: AdminThemeConfig = {
  // Основные цвета
  primary: '#your-color',
  primaryHover: '#your-hover-color',
  primaryActive: '#your-active-color',
  
  // ... остальные цвета
};

// Используйте новую тему
export const ADMIN_THEME: AdminThemeConfig = customAdminTheme;
```

### 3. Настройка отдельных цветов

Вы можете изменить любой цвет в существующей теме:

```typescript
export const ADMIN_THEME: AdminThemeConfig = {
  ...blackAdminTheme,
  primary: '#ff6b35', // Оранжевый основной цвет
  sidebarBackground: '#2c3e50', // Темно-синий сайдбар
};
```

## Структура цветов

### Основные цвета
- `primary` - Основной цвет кнопок и активных элементов
- `primaryHover` - Цвет при наведении
- `primaryActive` - Цвет при нажатии

### Сайдбар
- `sidebarBackground` - Фон сайдбара
- `sidebarBorder` - Границы в сайдбаре
- `sidebarActiveBackground` - Фон активного пункта меню
- `sidebarActiveBorder` - Граница активного пункта меню
- `sidebarActiveText` - Текст активного пункта меню
- `sidebarText` - Обычный текст в сайдбаре
- `sidebarTextHover` - Текст при наведении

### Текст и фон
- `background` - Основной фон
- `backgroundSecondary` - Вторичный фон
- `backgroundHover` - Фон при наведении
- `textPrimary` - Основной текст
- `textSecondary` - Вторичный текст
- `textMuted` - Приглушенный текст
- `textOnPrimary` - Текст на основном цвете

### Состояния
- `success` - Цвет успеха
- `warning` - Цвет предупреждения
- `error` - Цвет ошибки
- `info` - Информационный цвет

## Предустановленные темы

### Черная тема (текущая)
- Черный основной цвет (#000000)
- Черный сайдбар с белым текстом
- Современный минималистичный дизайн

### Синяя тема (оригинальная)
- Синий основной цвет (#2563eb)
- Белый сайдбар с темным текстом
- Классический дизайн

## Применение изменений

После изменения конфигурации:
1. Сохраните файл `admin-theme-config.ts`
2. Перезапустите приложение или обновите страницу
3. Изменения применятся автоматически ко всей админ панели

## Техническая информация

- Цвета применяются через CSS переменные в `getAdminThemeVariables()`
- Компоненты используют inline стили для максимальной совместимости
- Тема применяется автоматически при загрузке компонентов админ панели
