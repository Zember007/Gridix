# Решение проблем с конвертацией изображений

## Проблема

Deno Edge Functions не поддерживают `OffscreenCanvas` и имеют ограничения при работе с некоторыми форматами изображений через Web Image API.

## Решения

### Вариант 1: Использовать локальный скрипт для обработки (Рекомендуется)

Создайте Node.js скрипт, который обработает все изображения локально или на сервере:

```bash
# Установите зависимости
npm install sharp

# Запустите скрипт
node scripts/optimize-images-local.js
```

### Вариант 2: Использовать внешний API

Настройте Cloudinary, ImageKit или другой сервис для обработки изображений.

### Вариант 3: Использовать другую Edge Function платформу

Рассмотрите использование Vercel Edge Functions, Cloudflare Workers или AWS Lambda, которые имеют лучшую поддержку обработки изображений.

### Вариант 4: Обработать файлы пакетно в браузере

Используйте клиентский скрипт, который загрузит файлы, обработает их в браузере (используя Canvas API) и загрузит обратно.

## Для временного решения

Текущая функция обработает те файлы, которые может обработать, и пропустит проблемные с ошибками. Вы можете:

1. Просмотреть список ошибок из ответа функции
2. Обработать проблемные файлы вручную
3. Использовать внешний сервис для проблемных файлов
