Gridix Widget Examples

Explore different configurations of the Gridix projects widget.

1. Full Featured Widget

Complete widget with all features enabled (header, filters, full layout).

GridixWidget.init({ userId: 'demo-user-id', containerId: 'widget-full', height: '600px', width: '100%', compactMode: false, showHeader: true, showFilters: true });

2. Compact Widget

Compact version suitable for smaller containers and sidebars.

GridixWidget.init({ userId: 'demo-user-id', containerId: 'widget-compact', height: '500px', width: '100%', compactMode: true, showHeader: true, showFilters: true });

3. Minimal Widget

Minimal version with no header or filters - just the project grid.

GridixWidget.init({ userId: 'demo-user-id', containerId: 'widget-minimal', height: '400px', width: '100%', compactMode: true, showHeader: false, showFilters: false });

4. URL Parameters Configuration

Widget can also be configured via URL parameters for iframe embedding.

<iframe src="https://your-domain.com/widget.html?userId=demo-user-id&compactMode=true&showHeader=false&height=400px" width="100%" height="400" frameborder="0"> </iframe>

5. Integration Script

How to embed the widget in your website.

<!-- Include the widget script --> <script src="https://your-domain.com/widget.js"></script> <!-- Create container --> <div id="my-gridix-widget"></div> <!-- Initialize widget --> <script> GridixWidget.init({ userId: 'your-user-id', containerId: 'my-gridix-widget', height: '600px', width: '100%', compactMode: false, showHeader: true, showFilters: true, theme: 'light' }); </script>

Configuration Options

Available configuration options for the widget:

{ // Required - either projectId OR userId projectId?: string, // Show single project apartments userId?: string, // Show user's public projects gallery // Display Options compactMode?: boolean, // Enable compact layout (default: false) showHeader?: boolean, // Show header with title and view toggles (default: true) showFilters?: boolean, // Show filters section (default: true) // Container Options containerId?: string, // Custom container ID (default: 'gridix-widget-root') width?: string, // Container width (default: '100%') height?: string, // Container height (default: '600px') // Styling theme?: 'light' | 'dark' | 'auto', // Theme mode (default: 'light') cssUrl?: string, // Custom CSS URL // Localization lang?: 'ru' | 'en' | 'ka' | 'ar' // Language (default: auto-detect) }
