# Руководство по встраиванию виджета Gridix

## Введение

Виджет Gridix позволяет встроить интерактивные планировки недвижимости на ваш сайт. Виджет полностью изолирован от стилей вашего сайта и не создаёт конфликтов.

## Базовая установка (3 простых шага)

### Шаг 1: Добавьте контейнер

Добавьте `<div>` в место, где хотите видеть виджет:

```html
<div id="gridix-widget-root"></div>
```

### Шаг 2: Подключите скрипт

Добавьте скрипт виджета перед закрывающим тегом `</body>`:

```html
<script src="https://gridix.live/widget.js"></script>
```

### Шаг 3: Инициализируйте виджет

Добавьте код инициализации с ID вашего проекта:

```html
<script>
  window.GridixWidget.init({
    projectId: "ВАШ_ID_ПРОЕКТА",
    lang: "ru"
  });
</script>
```

## Полный пример

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Мой сайт с виджетом Gridix</title>
</head>
<body>
    <h1>Добро пожаловать на наш сайт</h1>
    <p>Посмотрите доступные квартиры:</p>
    
    <!-- Виджет будет здесь -->
    <div id="gridix-widget-root"></div>
    
    <!-- Скрипты виджета -->
    <script src="https://gridix.live/widget.js"></script>
    <script>
      window.GridixWidget.init({
        projectId: "05bc347c-e19a-41ba-8d87-1f7001d3329b",
        lang: "ru",
        width: "100%",
        height: "800px"
      });
    </script>
</body>
</html>
```

## Настройка виджета

### Основные параметры

```javascript
window.GridixWidget.init({
  projectId: "ВАШ_ID_ПРОЕКТА",  // Обязательно для одного проекта
  lang: "ru",                    // Язык: ru, en, ka, ar
  width: "100%",                 // Ширина виджета
  height: "800px"                // Высота виджета
});
```

### Показать несколько проектов

Если хотите показать все проекты пользователя:

```javascript
window.GridixWidget.init({
  userId: "ID_ПОЛЬЗОВАТЕЛЯ",
  lang: "ru",
  compactMode: true  // Компактный режим для списка
});
```

### Настройка внешнего вида

```javascript
window.GridixWidget.init({
  projectId: "ВАШ_ID_ПРОЕКТА",
  theme: "dark",          // Тёмная тема
  showHeader: false,      // Скрыть заголовок
  showFilters: false,     // Скрыть фильтры
  compactMode: true       // Компактный режим
});
```

### Кастомный контейнер

Если хотите использовать свой ID контейнера:

```html
<div id="my-custom-container"></div>

<script>
  window.GridixWidget.init({
    projectId: "ВАШ_ID_ПРОЕКТА",
    containerId: "my-custom-container"
  });
</script>
```

## Адаптация под мобильные устройства

Виджет автоматически адаптируется под размер экрана. Рекомендуем использовать:

```css
#gridix-widget-root {
  width: 100%;
  max-width: 1200px;
  margin: 0 auto;
}
```

```javascript
window.GridixWidget.init({
  projectId: "ВАШ_ID_ПРОЕКТА",
  width: "100%",
  height: "600px",  // Или используйте vh: "80vh"
  compactMode: true // Для мобильных
});
```

## Параметры через URL

Виджет поддерживает параметры через URL:

```html
<!-- Можно не указывать параметры в init() -->
<script src="https://gridix.live/widget.js"></script>
<script>
  window.GridixWidget.init();
</script>

<!-- А передать их через URL -->
https://your-site.com/?projectId=123&lang=ru&theme=dark
```

## Использование с популярными CMS

### WordPress

1. Установите плагин "Insert Headers and Footers"
2. В разделе Footer добавьте код виджета
3. На нужной странице добавьте HTML блок с контейнером

### Tilda

1. Добавьте блок "HTML-код" (T123)
2. Вставьте полный код виджета (контейнер + скрипты)

### Wix

1. Добавьте элемент "HTML iframe"
2. Вставьте код виджета

### Webflow

1. Добавьте Embed элемент
2. Вставьте код виджета

## Изоляция стилей

Виджет использует Shadow DOM, что гарантирует:

✅ **Нет конфликтов** - Ваши стили не влияют на виджет  
✅ **Стабильность** - Виджет выглядит одинаково везде  
✅ **Безопасность** - Виджет не может сломать ваш сайт  

Даже если у вас есть CSS вроде:

```css
* {
  margin: 0 !important;
  color: red !important;
}
```

Виджет всё равно будет работать корректно!

## Где взять ID проекта?

1. Войдите в панель управления Gridix
2. Откройте нужный проект
3. ID проекта указан в URL:
   - `https://gridix.live/projects/ВАШ_ID_ПРОЕКТА`

Или в настройках проекта в разделе "Встраивание".

## Частые вопросы

### Виджет не появляется

Проверьте:
1. Правильно ли указан `projectId`
2. Загрузился ли скрипт (проверьте в Network DevTools)
3. Нет ли ошибок в консоли браузера

### Виджет выглядит странно

Shadow DOM должен полностью изолировать стили. Если есть проблемы:
1. Проверьте версию браузера (нужен современный браузер)
2. Убедитесь, что не блокируете загрузку CSS
3. Проверьте консоль на ошибки

### Можно ли изменить стили виджета?

Базовые стили виджета изолированы. Для кастомизации:
1. Используйте параметры `theme`, `compactMode`, etc.
2. Свяжитесь с поддержкой для индивидуальных стилей

### Как отследить действия пользователя?

Виджет генерирует события, которые можно отслеживать:

```javascript
// В разработке - скоро будет доступно
window.addEventListener('gridix-apartment-selected', (e) => {
  console.log('Выбрана квартира:', e.detail);
});
```
